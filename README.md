## 목차
### ConvertDp 
- px과 db의 사이즈 convert
### ConvertPrice
- 소수점(Decimal)으로 단위 표기 1000 -> 1,000 
### ConvertTel
- 연락처 사이에 "-"를 지우거나 추가 기능
### DateFormatUtil
- 현재시간 조회, 시간 format 변경 기능
### FormatPass
- 비밀번호에 조건 부여 정규식 예시
### JsonConvertible
- Json 형식의 String값과 JsonObject의 값을 자유롭게 convert 가능
- DTO, VO등에 상속하여 동일하게 사용가능
### OnSingleClickListener
- 연속 클릭 방지 listener
### PreferenceManager
- Preferece 사용이 편리하도록 클래스로 제작
